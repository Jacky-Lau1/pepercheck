# Privacy Policy for Paper check

Last updated: July 3, 2026

Paper check is a Microsoft Edge extension that helps users check whether papers shown in Google Scholar search results already exist in their local literature index.

## Data Collection

Paper check does not collect, sell, or share personal information.

The extension works locally in the user's browser. Users may import a Markdown literature list or scan a local PDF folder to create a local paper index. This index may include paper titles, authors, years, DOI values, journal names, file names, and PDF metadata or visible PDF text needed for paper matching.

## Website Content

Paper check reads visible Google Scholar search result content only to compare paper titles with the user's local paper index and display local availability markers.

The extension may also work on user-approved Google Scholar mirror sites when the user manually grants permission for those sites.

## Local Storage

Extension settings, language preference, enable or disable status, temporary mirror-site settings, and the generated paper index are stored locally in the browser using extension storage.

Temporary data may be cleared when the browser session ends. Data saved by the user remains in the browser's local extension storage until the user clears it or removes the extension.

## Data Sharing

Paper check does not upload the user's literature index, PDF contents, browsing data, or Google Scholar search results to any external server.

Paper check does not transfer user data to third parties, except where required by law or with the user's explicit consent.

## Data Use

Paper check uses data only for its single purpose: checking whether Google Scholar search results match the user's local literature records.

Paper check does not use user data for advertising, profiling, credit decisions, lending purposes, or any purpose unrelated to local paper availability checking.

## Remote Code

Paper check does not load or execute remote JavaScript, WebAssembly, or external script files. All JavaScript files, CSS files, icons, and the bundled PDF.js library are included inside the extension package.

## Contact

If you have questions about this privacy policy, please contact the developer through the project's GitHub repository.
